Stage 1

サバイバル

まずこれだけ覚えて Neovim で生き残る

💾 保存・終了

N Spacew保存
N Spaceq閉じる
C :wq保存して閉じる
C :q!保存せず強制終了
💡 フォーカスが外れると自動保存される設定済み

🏃 移動（これだけ覚える）

h j k l← ↓ ↑ →
w / b次の単語 / 前の単語
e単語の末尾へ
0 / $行頭 / 行末
^行頭（空白除く）
gg / Gファイル先頭 / 末尾
{ / }段落ジャンプ ↑ / ↓
Ctrl+d / Ctrl+u半ページ ↓ / ↑
💡 数字+j/k で N 行移動（例: 10j）折り返し行も正しく動く設定済み

✏️ INSERT モードへの入り方

i / aカーソル前 / 後
I / A行頭 / 行末
o / O下に新行 / 上に新行
s1 文字消して INSERT
Cカーソル〜行末を消して INSERT

⚡ 基本編集

x1 文字削除
dd行削除（カット）
yy行コピー
p / P下に貼付 / 上に貼付
u / Ctrl+rUndo / Redo
.直前の操作を繰り返す
J下の行を結合
💡 d+移動 が基本。dw=単語削除、d$=行末まで削除

🔍 検索

/パターン前方検索
n / N次へ / 前へ
* / #カーソル上の単語を検索
N Escハイライト消去
:%s/old/new/gc確認しながら全置換

📁 ファイル操作

N Spaceffファイル検索 Telescope
N Spaceeエクスプローラ oil.nvim
N -親ディレクトリを開く oil.nvim
💡 oil.nvim ではディレクトリがバッファ。ファイル名編集 = リネーム、dd = 削除
Stage 2

効率化

編集速度を上げるテクニック

📦 テキストオブジェクト（Vim の文法）

vip段落を選択

🔄 よく使う操作の対応

Cmd+P ファイル検索→ Spaceff
Cmd+Shift+F 全文検索→ Spacefg
Cmd+S 保存→ Spacew
Cmd+W タブを閉じる→ Spacebd
Cmd+/ コメント→ gcc / gc
F12 定義へ→ gd
Shift+F12 参照→ gr
F2 リネーム→ Spacelr
Cmd+. クイックフィックス→ Spacela

🔄 よく使う操作の対応（続き）

ホバー→ K
サイドバー→ Spacee
タブ切替→ Shift+H / Shift+L
ペイン切替→ Ctrl+h/j/k/l
Git GUI→ Spacegg
フォーマット→ Spacecf（保存時自動）
Cmd+F ファイル内検索→ / or Space/
Cmd+D 同じ単語選択→ * → cgn → .
Cmd+Z / Cmd+Shift+Z→ u / Ctrl+r
