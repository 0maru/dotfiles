Stage 3

モード別操作

Copy / Setting / Search モードの詳細

📋 Copy モード — カーソル移動

C h j k l← ↓ ↑ →
C w / b / e次の単語 / 前の単語 / 単語末尾
C 0 / ^ / $行頭 / 行頭(空白除く) / 行末
C g / Gバッファ先頭 / 末尾
C H / M / L画面上端 / 中央 / 下端
C Ctrl+U / Ctrl+D半ページ上 / 下
C Ctrl+B / Ctrl+F1ページ上 / 下

✏️ Copy モード — 選択・ヤンク

C v文字単位選択
C V行単位選択
C Ctrl+V矩形選択
C yヤンク（コピーしてモード終了）

🔍 Copy モード — 検索・ジャンプ

C /検索パターンを入力（→ Search モード）
C n / N次のマッチ / 前のマッチ
C f{char}前方の文字へジャンプ
C F{char}後方の文字へジャンプ
C t{char}前方の文字の手前へ
C T{char}後方の文字の手前へ
C [ / ]前 / 次のセマンティックゾーン
C Esc / qCopy モード終了

🔎 Search モード

S Enter前のマッチへ（確定）
S Ctrl+N次のマッチへ
S Ctrl+P前のマッチへ
S Ctrl+Rマッチ方式を切り替え（文字列/正規表現）
S Ctrl+U検索パターンをクリア
S Esc検索を終了

⚗️ Setting モード（ペインリサイズ・透過度）

S h j k lペインリサイズ（1セル単位）
S 1〜9ペイン高さを 10%〜90% に設定
S Ctrl+1〜9ペイン幅を 10%〜90% に設定
S ;透過度を上げる（不透明に）
S -透過度を下げる（透明に）
S 0透過度をリセット
S Esc / qSetting モード終了
💡 Setting モードではキーを押し続けるとリピートされる。微調整に便利
